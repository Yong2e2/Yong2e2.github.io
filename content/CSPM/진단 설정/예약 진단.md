{: .no_toc }
<br>

기본메뉴에서 **예약진단** 메뉴를 클릭했을 때 나타나는 화면입니다. 등록된 진단대상이 있는 경우, 예약 진단 기능을 사용할 수 있습니다.

---

## 예약진단 화면 설명

<br>
<center>
    <img
        src="/assets/images/예약진단 화면설명.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>왼쪽 체크박스 클릭 시 버튼 활성화</figcaption>
</center>
<br>
예약진단 페이지 내 각 메뉴에 대한 설명은 다음과 같습니다.

| 번호 | 설명                                                                                                                                                                                                                                                                                                  |
| ---- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1    | 예약 진단을 생성된 **별칭**이 표시됩니다.                                                                                                                                                                                                                                                             |
| 2    | 예약진단이 시작되는 **날짜**가 표시됩니다.                                                                                                                                                                                                                                                            |
|      | **(예약진단이 처음으로 생성된 경우, 다음날 00시 이후로 동작합니다.)**                                                                                                                                                                                                                                 |
| 3    | 예약진단이 종료되는 날짜가 표시됩니다.                                                                                                                                                                                                                                                                |
| 4    | 반복할 예약 타입을 지정합니다. 매일, 매주, 매월의 옵션이 존재하며, 매일의 경우 매일 지정한 시간에 진단이 시작되며, 매주의 경우 특정 요일과 시간을 선택하여 매주 지정한 요일과 시간에 진단이 시작됩니다. 매월의 경우 진단을 시작할 날짜와 시간을 입력하여 매달 지정한 날짜와 시간에 진단이 시작됩니다. |
| 5    | 반복 예약 타입이 **매월일 경우, 지정된 날짜**가 표시되고, 반복 예약 타입이 **매주일 경우 지정된 요일**이 표시됩니다.                                                                                                                                                                                  |
| 6    | 예약 진단이 실행되는 **시간**이 표시됩니다.                                                                                                                                                                                                                                                           |
| 7    | 예약진단을 생성한 **사용자**를 표시합니다.                                                                                                                                                                                                                                                            |
| 8    | 예약진단을 **생성**합니다.                                                                                                                                                                                                                                                                            |
| 9    | 설정된 예약진단을 **수정**하거나 **삭제**합니다.                                                                                                                                                                                                                                                      |

## 예약 진단 추가

<br>

예약진단 페이지에서 <img src="/assets/images/button/생성버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼을 클릭 시 진단 스케줄을 등록하는 창이 표시됩니다. 기본정보에서 주기를 설정하고 진단 대상에서 진단하고싶은 키를 선택 후 <img src="/assets/images/button/저장버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;">버튼을 누릅니다

<center>
    <img
        src="/assets/images/예약진단_생성.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>예약진단 생성</figcaption>
</center>
<br>

| 항목 | 설명                                              |
| ---- | ------------------------------------------------- |
| 반복 | 매일, 매주 매월을 선택 가능하며, 반복할 주기 선택 |
| 시간 | 진단 스케줄이 반복될 시간 입력                    |
| 기간 | 해당 스케줄이 유지되는 기간 입력                  |

<br>
<br>

입력한 정보를 확인한 후, <img src="/assets/images/button/저장버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼을 클릭하고 <img src="/assets/images/button/예스버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼을 클릭하여 지금까지 입력한 정보를 저장합니다.

<center>
    <img
        src="/assets/images/예약진단_생성_확인팝업.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>예약진단 저장 시 출력되는 메시지</figcaption>
</center>
<br>

예약진단메뉴에서 생성한 예약진단을 확인할 수 있고 <img src="/assets/images/button/작업버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼을 통해 예약진단을 수정하거나 삭제할 수 있습니다.

    <figcaption>생성한 예약진단 확인</figcaption>
</center>
