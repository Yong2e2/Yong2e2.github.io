## 사용자 관리

<br>
<center>
    <img
        src="/assets/images/디클로설정_사용자 권한 관리.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>사용자 관리 화면</figcaption>
</center>

기본메뉴에서 D-CLO 설정 > **사용자 관리**를 클릭했을 때 표시되는 화면입니다. **사용자를 새로 등록**할 수 있으며, 위에서 **생성한 권한을 부여**할 수 있습니다. 사용자가 최초로 생성 될 시, 사용자를 생성 할 때 입력하는 **이메일로 최초 비밀번호가 발송**되므로, 정확하게 입력하여야 합니다.

<br>

## 사용자 추가

<br>
<center>
    <img
        src="/assets/images/디클로설정_사용자관리_생성.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>사용자 생성 – 1 (사용자 생성 버튼)</figcaption>
</center>
사용자 추가가 필요할 경우, 왼쪽 상단의 <img src="/assets/images/button/생성버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼을 클릭합니다.

<br>
<center>
    <img
        src="/assets/images/디클로설정_사용자관리_입력.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>사용자 생성 – 2 (추가할 사용자 정보 입력)</figcaption>
</center>
<img src="/assets/images/button/생성버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼을 클릭 시 위 사진처럼 사용자를 추가하는 화면이 표시됩니다. 로그인 할 아이디, 이름, 연락처, 이메일, 상태, 사용자의 권한, 회사명은 필수로 입력하셔야 하며, 이메일의 경우, 최초 로그인 패스워드가 전송되므로 정확하게 입력해주셔야 합니다.

<br>
<center>
    <img
        src="/assets/images/디클로설정_사용자관리_생성팝업.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>사용자 생성 – 3 (사용자 생성 팝업)</figcaption>
</center>
사용자가 저장되었다는 팝업을 확인할 수 있으며, 등록 시 입력한 이메일에서 해당 사용자의 초기 비밀번호와 OTP 정보를 확인할 수 있습니다.

</center>

## 사용자 수정

<br>
<center>
    <img
        src="/assets/images/디클로설정_사용자관리_수정.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>사용자 수정 버튼 위치</figcaption>
</center>
사용자의 정보와 사용자 권한의 수정이 필요한 경우 해당 사용자를 체크 후 왼쪽 상단의 <img src="/assets/images/button/작업버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼 -> <img src="/assets/images/button/수정버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼을 클릭하여 정보를 수정할 수 있습니다.

## 비밀번호 초기화

<br>
<center>
    <img
        src="/assets/images/디클로설정_사용자관리_초기화.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>사용자 수정 버튼 위치</figcaption>
</center>
사용자의 비밀번호 초기화가 필요한 경우 해당 사용자를 체크 후 왼쪽 상단의 <img src="/assets/images/button/작업버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼 -> <img src="/assets/images/button/비밀번호초기화버튼.png" alt="description" style="vertical-align:middle width:15px; height:15px;"> 버튼을 클릭하여 비밀번호 초기화를 진행할 수 있습니다.
