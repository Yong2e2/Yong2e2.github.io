생성 페이지에서 토큰 이름, 설명, 사용자 연결, 만료일을 선택할 수 있습니다.
<br>
만료일은 7, 30, 60일을 기본으로 제공하며 사용자 설정으로 최대 365일까지 설정 가능합니다.

## Open API Token 목록

<center>
    <img
        src="/assets/images/OpenAPIToken_목록.png"
        width="1600"
        height="900"
        style="border: 2px solid black;"
    />
    <figcaption>Open API Token 목록</figcaption>
</center>

| 번호 | 설명                                                                     |
| ---- | ------------------------------------------------------------------------ |
| 1    | 생성한 토큰을 선택합니다.                                                |
| 2    | 토큰의 상세 정보를 확인할 수 있습니다.                                   |
| 3    | 토큰값을 복사하여 사용할 수 있습니다.                                    |
| 4    | 사용한 토큰인지 테스트를 위해 Credential 입력란에 붙여넣기를 진행합니다. |
| 5    | 버튼을 눌러 Open API 서버에 요청을 시도합니다.                           |
